概要

Memory Map ~Travel~は、旅行の思い出を地図上にビジュアルに記録できる革新的なアプリです。 写真をパズルのピースのように切り取り、訪れた場所にマッピングすることで、 あなただけの思い出マップを作成できます。

47都道府県、北海道179市町村、東京都62市区町村の詳細な地図データを収録し、 日本全国の旅の記録を美しく保存できます。

主な機能

  • パズル形式の写真マッピング

    写真を地図の形に切り取って思い出を記録

  • 詳細な地図データ

    47都道府県と主要市町村の地図を収録

  • マルチプラットフォーム対応

    iOS、iPadOS、macOS、visionOSで利用可能

  • 思い出の共有

    作成したマップをSNSで簡単にシェア

  • オフライン利用

    インターネット接続なしでも利用可能

開発ストーリー

このアプリは、「旅行の思い出をより視覚的に、より楽しく残したい」という想いから生まれました。 従来の写真アルバムでは表現できない、訪れた場所と思い出の結びつきを表現する新しい方法を模索し、 パズルのピースとして写真を地図に配置するというアイデアに辿り着きました。

開発では特に、ユーザーが直感的に操作できるインターフェースの設計と、 大量の地図データを効率的に扱うための最適化に注力しました。 結果として、80,000人以上のユーザーに愛用いただけるアプリとなりました。
